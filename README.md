# README Proyecto 1

## Descripción
El objetivo del juego es matar los 4 patos en menos de 20 segundos.

## Instrucciones para jugar
Debes disparar a los patos haciendo **click izquierdo** con el ratón.  
El objetivo es eliminar los **4 patos antes de que se acabe el tiempo (20 segundos)**.
